# Product Definition

## Initial Concept
To develop a high-performance CFD solver for porous media using SDFs and IBM, accessible via Python.

## Primary Purpose
The primary purpose of this project is to develop a robust and efficient Computational Fluid Dynamics (CFD) solver tailored for simulating fluid flow within porous media. This solver will leverage Signed Distance Functions (SDFs) to accurately represent complex geometries and incorporate the Immersed Boundary Method (IBM) for effective handling of fluid-solid interactions. The ultimate goal is to provide a high-performance simulation tool that is readily accessible and controllable through a Python interface.

## Target Users
The primary target users for this CFD solver are researchers and engineers working in both academia and industry. This includes specialists in fields such as porous media flow, hydrogeology, materials science, and chemical engineering, who require advanced computational tools for their simulation and analysis needs.

## Core Features
The CFD solver will offer the following core features and functionalities:
- **High-Performance Fluid Flow Simulations:** Capable of simulating fluid flow in complex geometries defined by SDFs, specifically optimized for porous media applications.
- **Immersed Boundary Method (IBM) Integration:** Seamless integration of the Immersed Boundary Method to accurately and efficiently handle fluid-solid interactions within complex geometries.
- **Python API:** A comprehensive and user-friendly Python API that enables flexible simulation setup, execution, and advanced post-processing workflows, making the solver accessible to a broader range of users and facilitating integration into existing research pipelines.
